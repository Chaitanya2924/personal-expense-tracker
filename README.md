# Personal Expense Tracker

## Description
This is a console-based Personal Expense Tracker built using Python.
It allows users to record daily expenses, categorize them, set a monthly budget,
and track spending effectively.

## Features
- Add expenses with date, category, amount, and description
- Set and store a monthly budget
- View all expenses
- View monthly expenses
- Budget warning when limit is exceeded
- Data persistence using CSV files

## Concepts Used
- Python basics
- Lists and dictionaries
- Conditional statements and loops
- Functions
- File handling (CSV)

## How to Run
1. Run the expense_tracker.py file
2. Choose options from the menu
3. Data is saved automatically
